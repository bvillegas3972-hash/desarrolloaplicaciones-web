## Archivos del proyecto

- `index.html`: contiene la estructura de la interfaz.
- `styles.css`: contiene el diseño visual, colores, distribución y adaptación a pantallas pequeñas.
- `script.js`: contiene la lógica de la calculadora.

## Funciones implementadas

- Suma, resta, multiplicación y división.
- Botón de limpiar `AC`.
- Botón para borrar el último dígito `DEL`.
- Números decimales.
- Cambio de signo `±`.
- Porcentaje `%`.
- Uso con mouse y teclado.
- Validación para evitar división entre cero.
